<h1>Username List Generator</h1>

_"We have username anarchy at home."_<br>
_Username anarchy at home:_
<br><br>
Simple script to generate lists of usernames from a list of First [Middle] Last names.

```
Generates a list of usernames from a list of names.

positional arguments:
  input                 Text file with names.
  output                Name of resutling file.

options:
  -h, --help            show this help message and exit
  --middlenames, -m     Process middle names / two last names.
  --go_crazy, -g        Include non-standard combinations.
  --case_sensitive, -c  Add capitalization variants. All lowercase if disabled.
```
